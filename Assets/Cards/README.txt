How To Create a New Card (WIP):
1. Right Click in the Project Window, Select Create, Select Cards.
2. Fill out the Card's data in the window on the right.
3. Drag this into Assets>Cards>Card Data (This is just for organization)
4. In the Cards folder is a CardTemplate. Drag it into the Scene. Rename the "CardTemplate" To the name of the card.
5. Go to Card Data and drag the corresponding data into the Card slot with the template selected in the scene.
6. Under the Card Template in the scene are several canvas items, drag each corresponding item into the slot in the component on the right. (I.e. drag Name into Name Text)
7. Select Artwork.
8. DONE!
